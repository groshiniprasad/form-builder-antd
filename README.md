## Generic Form Builder
This works similarly to the Google forms 
#### Features
- Add/Delete Elements
- Duplicate ELements
- Set input required
- Drag elements to reorder
- Multiple element types
    - TextInput
    - TextArea
    - RadioInput
    - NumberInput
    - DateInput
    - TimeInput
    - DropDown 

#### Usage
```bash
git clone {GIT_URL}
cd {PROJECTNAME}
npm install
npm start
```
